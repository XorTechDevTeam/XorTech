Подробное описание

Нулевой модуль.

Предоставляет основные функции общего назначения в скриптовой подсистеме. Помимо этого, не делает практически ничего, лишь занимет нулевой индекс в менеджере модулей. Кроме того, служит примером для разработки простых модулей без межмодульных зависимостей.
